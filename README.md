# MangaAutoColor Pro v3.0.1

## Início Rápido

```bash
# 1. Instalar dependências e baixar modelos (SD 1.5 + ControlNet)
scripts\windows\install.bat

# 2. Iniciar servidor API (Localhost:8000)
scripts\windows\run.bat
```
